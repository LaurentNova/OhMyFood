# OhMyFood

Projet OpenClassrooms consistant à intégrer en mobile-first un site de réservation de menus pour des restaurants.

## Objectifs du projet
- Intégration HTML et CSS en respectant la maquette.
- Utilisation de Sass pour organiser les styles.
- Mise en place d'animations (loader, coeur, coche verte).
- Respect des bonnes pratiques et validation W3C.

## Fonctionnalités
- Page d’accueil listant les restaurants.
- Pages menus avec animations.
- Loader animé.
- Coche verte animée sur les plats.
- Design responsive mobile-first.

## Technologies
- HTML5  
- Sass / CSS3  
- Font Awesome  
- Google Fonts  

## Structure (simplifiée)
- `index.html` : page d’accueil  
- `restaurants/` : pages des restaurants  
- `scss/` : fichiers Sass  
- `assets/` : images + CSS compilé
